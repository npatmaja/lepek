# lepek

_lepek is a template project for Express CoffeeScript.

# Requirements

* NodeJs
* Express
* ECT template engine
* Stylus
* Mocha
* Chai

# Installation
To use this template, run the following commands:
```
git clone https://github.com/npatmaja/lepek.git [project name]
cd [project name]
npm install
npm install mocha -g
```
